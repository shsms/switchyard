    Switchyard reference

    Topology canvas

    • Click a node — inspect it (charts + overrides + knobs in the side panel).
    • Ctrl/Cmd+Click — toggle a node into the multi-selection. Drag on empty canvas — rubber-band a multi-selection.
    • Right-click — context menu. With a selection: Copy, Cut, Delete. With nothing selected and a non-empty clipboard: Paste. Right-clicking a node not in the current selection resets the selection to that node first.
    • Shift hold — drag from one node to another to create an edge.
    • Ctrl/Cmd+C — copy selection. Ctrl/Cmd+V — paste. Ctrl/Cmd+X — cut. Each paste re-uses the last copy; clipboard captures structure only (categories + internal edges), so clones use category defaults.
    • Del / Backspace — delete selection. Ctrl/Cmd+A — select all visible. Esc — deselect.

    Side panel + chrome

    • Add component (lower side panel) — pick a category, click Create; an auto-id'd component appears. Wire it via Shift-drag.
    • Defaults — per-category defaults (Lisp values); edits ride the pending log.
    • Scenarios — save the current pending log as a named recipe; load to replay it.
    • Overrides pill — total active overrides (on disk + unsaved). * means unsaved. Click to inspect / × individual entries.
    • Persist — flush pending edits + honored × marks to config.ui-overrides.<microgrid-id>.lisp.
    • Discard — drop the pending log + × marks; reload from disk.

    REPL

    • Ctrl/Cmd+Enter — eval. Tab — complete.
    • Drag the splitter between the topology row and the drawer to resize.
